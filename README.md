# real-time-mobile-robot-control-using-hand-gestures
 
 Đề tài viết trên ngôn ngữ python, sử dụng Raspberry Pi và Arduino Wifi ESP826 WeMos D1, điều khiển mobile robot di chuyển theo cử chỉ tĩnh của bàn tay.
